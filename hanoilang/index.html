Hanoi is a programming language that is

  • concise. Only frequently used data types are included, namely string, integer, real, logical, and functional.
  • intuitive. A data matrix may have row names and column names. A map contains variables of different types. Functions are with named parameters, and thus no need for comprehensive documentation.
  • scalable. Your code is ready for big time, i.e. easily translated to C/C++. Matrix indices start from 0. Concurrent programming constructs mean fewer occasions when 11 out of 12 cores sitting idle leaving one doing all the hard work.

The syntax should be familiar to most programmers. Have a look at some examples, download the console, and you are ready to go.

Identifiers

  1. "" | `` string literals
    • "hello, world"
    • `raw\tstring` // == "raw\\tstring"
  2. integer literals (base10, C++ syntax)
    • 2016; -1
  3. real numbers, floating point literals (C++ syntax)
    • 3.14; -1.0; 1e06; .5
  4. true | false logical values
    • 0 > 1 // == false
  5. @() functions
    • @(a, b = 1) a + b
  6. if | else | for | do | while| continue | break | return flow controls
    • for (a : ["Hello", "world"]) print (a)
  7. variable names containing alphanumeric characters or "_", and not starting with a digit.
    • a = sqrt(b0 * b0 + _c * _c)

Operators

  1. unary || parallel execution
  2. = assignment
  3. compound assignments
    • += plus, -= minus, *= multiply, /= divide, %= remainder, &= and, |= or, ^= reshape, ^^ colnames, << rownames
  4. branching
    • ?, ternary conditional, string substitute
    • ?, ~ case switch
  5. . map reference
  6. ; row bind
  7. , column bind, map concatenate
  8. matrix manipulation
    • # matrix replicate
    • ^ matrix reshape, string match capture
  9. || short-circuit or
  10. && short-circuit and
  11. | logical or, string match
  12. & logical and, string intersection
  13. == equal, != not equal
  14. > greater , >= greater or equal, < less than, <= less than or equal
  15. : sequence
  16. addition, subtraction
    • + numeric addition, string at
    • - numeric subtraction, string substring
  17. multiplication, division
    • * numeric multiplication, string join
    • / numeric division, string split
    • % integral remainder, string grep
  18. unary operators
    • + numerical sum, string concatenate, logical any
    • - numerical negate, string reverse
    • * string length, logical all
    • ! string trim, logical negate
  19. ' transpose
  20. () function call, [] slice, [~;~] slice with negation
  21. unary matrix operators
    • < row names
    • ^ column names
    • # number of elements
    • & as string
    • % as integer
    • / as real
