Heatmap & clustering

I've created a number of convenient functions to make data analysis easy to perform in R. If you are familiar with functions in Excel, you should be able to perform the provided analyses in R.

The first step is to make the scripts available to R. Open an R console and enter

source("http://www.hanoilang.org/tp.r")

Your data table should be in plain tab-deliminated text format, with a single header line. Once you are more familiar with R, you will be able to handle a variety of data formats. But if you are a novice, use other tools, i.e. Excel, to create the tab-deliminated text file.

Enjoy!
