Beta-binomial test

Thang V Pham

16 April 2020

Installation

You will need to install the package ibb from the site https://tvpham.github.io/ibb/.

On Windows with a recent version of R, the following is likely to work

install.packages("https://tvpham.github.io/ibb/ibb_18.09.zip", repos = NULL, type = "win.binary")

Like other R packages, you will need to install the package once only.

Two-group comparison

Load the ion library and load an example dataset (or pointing to your local, tab-delimitated data file)

source("https://tvpham.github.io/ion.r")

d <- ion$load("https://tvpham.github.io/data/example-3groups.txt")

Check the data with the head function

head(d)
##    a1  a2  a3  b1  b2  b3  c1  c2
## 1 624 496 509 414 394 375 325 288
## 2 615 854 930 341 523 360 359 329
## 3 553 560 745 819 490 481 480 500
## 4 525 412 401 354 321 310 258 228
## 5 484 284 315 268 282 307 270 298
## 6 482 348 400 242 365 367  81 118

To compare the a samples to b samples, call

out <- ion$beta_binomial_2g(d,
                            c("a1", "a2", "a3"),
                            c("b1", "b2", "b3"))

ion$save(out, "output.txt")

In the first statement, the first parameter is the raw count data. The second parameter is column names of the first group and the third parameter column names of the second group. The output has three components “fc” fold change, “pval” p-value, and “pval.BH” p-value adjusted for multiple testing using the Benjamini-Hochberg method (Benjamini & Hochberg, 1995).

The second statement write the result to an output file. Nevertheless, we often want to write out the normalized count data as well. This is because the beta-binomial test uses the normalized values internally. The normalized values are obtained by scaling then original count values so that the total count are equal across all samples.

d_norm <- ion$normalize_global(d)

ion$save(cbind(d, d_norm, out), "output.txt")

Three-group comparison

To compare a, b, and c samples, call

out <- ion$beta_binomial_3g(d,
                            c("a1", "a2", "a3"),
                            c("b1", "b2", "b3"),
                            c("c1", "c2"))

The output now contains only two components: “pval” “pval.BH” because there is no fold change among three groups.

There is also a function for four-group comparison ion$beta_binomial_4g(). Refer to the manual for comparison of more than four groups.

Paired sample test

If a and b samples are matched, for example individual patients before and after treatment, a paired sample test is appropriate

out <- ion$beta_binomial_2g_paired(d,
                                   c("a1", "a2", "a3"),
                                   c("b1", "b2", "b3"))

Like the independent sample test, the result of a paired test has three components: “fc”, “pval”, and “pval.BH” standing for fold change, p-value and p-value corrected for multiple testing using the Benjamini Hochberg method, respectively.
